CSV Format Requirements

Your CSV file must have the following columns (in order):

1 Customer Name (required)
2 Mobile Number (required)
3 Shipping Address (required)
4 Product ID/Code (required)
5 Quantity (required)
6 Price Per Unit (AED) (required)
7 Order Code (optional)
8 Product Variant (optional)
9 Notes (optional)
10 Product Link (optional)
11 Order Date (YYYY-MM-DD) (optional)

Important Notes

  • If Order Code is empty, a new code will be auto-generated
  • Customer name, mobile number, shipping address, and product ID are required
  • If Product ID is not found, the order will be created without a product link
  • Quantity must be greater than 0
  • Price cannot be negative
  • Product Variant will be added to order notes as "Product Variant: [value]"
  • Orders with existing order codes will be skipped
