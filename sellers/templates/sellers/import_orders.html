CSV Format Requirements

Your CSV file must have the following columns (in order):

1 Customer Name (required)
2 Mobile Number (required)
3 Shipping Address (required)
4 Product ID/Code (optional)
5 Quantity (required)
6 Price Per Unit (AED) (required)
7 Order Code (optional)
8 Product Variant (optional)
9 Notes (optional)
10 Product Link (required)
11 Order Date (YYYY-MM-DD) (optional)

Important Notes

  • If Order Code is empty, a new code will be auto-generated
  • Customer name, mobile number, shipping address, and product link are required
  • Product Link is required - must be a valid URL to the product
  • Product ID/Code is optional - if not found, order will be created without product link
  • Quantity must be greater than 0
  • Price cannot be negative
  • Product Variant: For single variant, enter the variant name. For multiple variants, separate them with commas (e.g., "Red Small, Blue Medium, Green Large") - each variant will create a separate order
  • Orders with existing order codes will be skipped
  • All orders will be assigned to your seller account
